Página de inicio de Valvula

¡Hola! Valvula es un servidor de políticas OpenSource de alto rendimiento para Postfix, escrito en C, y que proporciona soporte para descrepancia de lgin de envío, cuotas de correo, listas blancas y listas negras a nivel de dominio y de usuario, y mucho más..

Es amigable para administradores de sistemas porque proporciona muchas características automáticas (como detectar automáticamente las bases de datos MySQL para ser capaz de detectar los dominios locales y/o cuentas locales para poder tomar mejores decisiones).

Valvula es totalmente extensible a través de plugins y está compuesto por una librería base (libValvula) que se integra en un servidor listo para usar (valvulad) que es capaz de ejecutar en diferentes puertos con diferentes módulos de manera que pueda conectar el mismo proceso valvula en diferentes puntos dentro de las secciones de restricción de Postfix.

Valvula está licenciado bajo los términos de la GPL 2.0 y está siendo usado, entre otros, por Core-Admin para proporcionar gestión avanzada de políticas de correo junto con sistemas de tickets para envíos masivos.

Acerca de Aspl

Aspl (Advanced Software Production Line) es una empresa española que proporciona soporte y servicios para introducir productos open source dentro de las organizaciones, especialmente GNU/Linux.

Fundada en el 2000, su principal línea de desarrollo es proporcionar soporte para el desarrollo de software así como soporte para instalar servidores GNU/Linux que proporcionan distintos servicios.

Visite nuestra página en http://www.aspl.es para saber más sobre nosotros.
