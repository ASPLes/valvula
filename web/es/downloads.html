Últimos paquetes debian / ubuntu

Use los siguientes repositorios APT para instalar Valvula y sus dependencias.

NOTA: no use estas fuentes si está utilizando Core-Admin a menos que sepa lo que está haciendo.

  # public aspl ubuntu repository (XENIAL lts 16.04)
  deb http://www.aspl.es/ubuntu/public xenial/

  # public aspl ubuntu repository (PRECISE lts 12.04)
  deb http://www.aspl.es/ubuntu/public precise/

  # public aspl debian repository (WHEEZY)
  deb http://www.aspl.es/debian/public wheezy/

  # public aspl debian repository (SQUEEZE)
  deb http://www.aspl.es/debian/public squeeze/

  # public aspl debian repository (LENNY)
  deb http://www.aspl.es/debian/public lenny/

Luego ejecute un apt-cache search valvula para buscar paquetes Valvula.

Paquetes estables para CentOS

Soporte de paquetes para CentOS se proporciona a través de repositorios yum:

NOTE: no use estas fuentes si está utilizando Core-Admin a menos que sepa lo que está haciendo.


  # /etc/yum.repos.d/ASPL-Base.repo
  # Aspl-Public repo 
  #
  # Paquetes de ASPL para libaxl, vortex, nopoll, myqtt, ext-dns, valvula
  # y otros proyectos opensource
  #
  
  [aspl-public]
  name=Aspl-Public-$releasever - Base
  baseurl=http://www.aspl.es/centos/public/$releasever/$basearch/
  gpgcheck=0
 
  # Aspl-Private repo 
  #
  # Paquetes de ASPL para proyectos comerciales como core-admin
  #
  [aspl-private]
  name=Aspl-Private-$releasever - Base
  baseurl=http://www.aspl.es/centos/private/$releasever/$basearch/
  gpgcheck=0

Luego ejecute yum search valvula para buscar paquetes de Valvula.
